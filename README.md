# LAN Clipboard Sync

LAN Clipboard Sync is a local-network clipboard sharing app for Android and Windows.

It lets devices on the same Wi-Fi find each other, connect into a small sharing group, and send clipboard content without using a cloud service.

## Features

- Discover nearby Android and Windows devices over LAN.
- Connect devices into one clipboard sharing group.
- Send and receive copied text.
- Send and receive copied images, screenshots, and image files.
- Auto-copy received clipboard content when enabled.
- Manually accept or reject incoming connection requests.
- Optional auto-accept for trusted local use.
- Clipboard history for recent sent and received items.
- Windows companion app with tray support and single-instance behavior.
- Firewall helper for Windows LAN access.

## Notes

- Devices must be on the same reachable Wi-Fi/LAN.
- Some public, guest, or isolated networks block device-to-device traffic.
- Android background clipboard access is limited by the OS, so automatic sending works best while the app is open.
- Clipboard sync supports text and images only; other files are ignored.
